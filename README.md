byronbuckley.com
================

## Pre-requisites
- Homebrew
- Ruby
- Jekyll
- Node 8

## Get Started
Install [Jekyll][3]

Installation issues? Try
- Updating [Jekyll][5],
- This [Gist][4]

## Run
```
# install dev dependencies
npm i

# dev
npm run dev
```

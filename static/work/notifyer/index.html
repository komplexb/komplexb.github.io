A AWS Lambda function that uses Microsoft’s OneNote as a data source, Notifyer utilizes OneNote’s API to retrieve a random note from a given section.

Use case?

  • Get a daily quote from your curated collection.
  • Effortlessly resurface a buried quote and re-apply it to the present.
